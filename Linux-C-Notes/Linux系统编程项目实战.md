# 项目前瞻

基于IPV4的流媒体广播系统。



## 项目需求

需要基于客户机和服务器模型的网络音频广播/点播系统。

用于语音教室和公共广播。

分为服务器和客户机，服务器在PC上运行，客户机在PC机或者嵌入式设备。

服务器可以向局域网内以多播的方式给所有客户机发送数据，客户机自己选择接收数据。
