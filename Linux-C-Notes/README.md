## 环境搭建步骤

### 一、拥有一个linux系统
- 虚拟机
- WSL(Windows Subsystem for Linux)
- 云服务器
- 双系统

### 二、安装编译环境

- 编译器：`gcc version 12.2.0 (Debian 12.2.0-14)`
- 代码提示：`clangd-vscode`
- 代码格式化：`clang-format`
- 调试工具：`gdb`
- 版本控制：`git`
- 编辑器：`vscode`
- AI补全：`Fitten Code`

### 三、下载仓库
- `fork`并且`clone`到本地（推荐）
- 直接下载压缩包到本地解压
